Configuration & Usage

Simply define a <mindmap> view in your XML or use widget="mindmap" on One2many fields.
Fully compatible with Odoo 17 & 18 (Enterprise & Community).

<mindmap parent_field="parent_id" child_field="child_ids">
<field name="name"/>
</mindmap>
