How to Configure

Standard XML Definition

<mindmap parent_field="parent_id" child_field="child_ids">
    <field name="name"/>
    <templates>
        <t t-name="mindmap-box">
            <div class="fw-bold"><field name="name"/></div>
        </t>
    </templates>
</mindmap>
Standalone View

Best for full-screen analysis of entire record sets.

action="action_view_mindmap"
Form Widget

Best for managing sub-records within a parent.

widget="mindmap"
