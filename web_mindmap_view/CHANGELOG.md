# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-02-05

### Added

- **Interactive Mind Map View**: Infinite panning, zooming, and node selection.
- **One2many Field Widget**: Support for managing recursive records directly inside Form Views (`widget="mindmap"`).
- **Side Panel Inspector**: Integrated right-side panel for editing node details without context switching.
- **Dark Mode**: Fully optimized dark theme with high-contrast elements.
- **Search & Highlighting**: Find nodes instantly with path highlighting.
- **Dynamic Configuration**: Support for `side_panel_fields` and custom XML templates.

### Fixed

- Addressed layout issues where the view would not take full height in Field Widget mode.
- Fixed auto-update behavior when creating or deleting records via the Field Widget.
